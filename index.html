Flagship Product

Derech Journey Planner

A multi-stop journey planner and navigation MVP designed like a modern route intelligence product. It supports complete whole-journey planning, route portfolio selection, active navigation, stop handling, mid-trip overrides, and re-planning remaining legs without breaking completed ones.

  • Designed as a map-first React and TypeScript application with bottom-sheet interactions
  • Supports multi-stop trip building, traffic alerts, route comparisons, and live journey states
  • Demonstrates product thinking, UI architecture, and complex state management for frontend systems

An ongoing research project focused on building a federated high-performance computing cluster for collaborative AI workloads using SLURM for job scheduling and resource management.

  • Benchmarked performance, scheduling efficiency, and job prioritization
